Intelligent Adaptive Website using Neuro Feedback

Intelligent

Intelligence is the ability to aquire and apply knowledge. For example, when we eat chocolate, our brain says it taste good; so next time when we see chocolate, we will want to eat it again. Intelligence in the context of an adaptive website is the same.

An example can be when you visit a website and its color theme hurts your eyes, you will be reluctant to visit that site often. Now if the website can know you do not like the color theme, it can change its color theme to make your browsing experience better.

Adaptive

Adaptive. Always changing. A lot of variables account for our likings. In the morning we may like heavy animation and bright colors, but at night we may prefer light animation and harmonious colors. Emotional states can also account for changes in preferences and the adaptive website aims to adapt to changes through out the day.

Website

We are all familar with websites. This is a website. It is a way to get and display information from a server so that you can view it in your browser. Internet connects servers to computers and smartphones world wide so the same website can be accessed from around the world.

Websites keep evolving and keep on getting more interactive with a goal of making a style that is appealing. One problem with this appoach is that people have different tastes for websites. A person may like minimal website while another may like websites with animations. An intelligent adaptive websites aims to change the website automatically to user's liking.

Neurofeedback

Feedback is a major factor in learning. There has to be some way of the users telling the website what they like. This is done with a comercial dry electroencephalogram device. The device captures a blurred form of brain activity and together with a program that sorts brain activity into a quantifiable feedback. The system can be called a non-intrusive brain machine interface. The sorting program, in this case a simulated neuro network, will tell the website to change to a different color or style.
